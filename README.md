## Running the app

```bash
    $ git clone https://github.com/Thagana/couchbuddies.git
```

```bash
    $ cd couchbuddies
```

```bash
    $ yarn
```

### for android

```bash
    $ react-native run-android
```

### for iOS

```bash
    $ react-native run-ios
```
